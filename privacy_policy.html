Privacy Policy for Superstar Investors Tracker

This Chrome extension helps users discover common stock picks held by top investors like Warren Buffett, Rakesh Jhunjhunwala,Ray Dalio and many more.

What Data We Collect

  • We do not collect any personally identifiable information.
  • All scraped data (investor names, stock tickers, and prices) is stored locally in your browser and is never transmitted to any external server.
  • Data is automatically cleared when the user downloads, copies, or clears it manually.

How We Use Data

The data stored is used solely to enhance user experience and allow users to analyze investor portfolios efficiently. It is never shared or sold to third parties.

Permissions

The extension uses Chrome's built-in APIs to access the current tab, store scraped data locally, and notify users when an investor has already been scraped.

Your Consent

By using this extension, you consent to this privacy policy.
